# 使用 Remax 编写的仿闲鱼UI的微信小程序

## 计划编写：

* 首页
* 会玩
* 消息
* 我的

## 运行方式

```text
// 项目拷下来后，先下载依赖
npm install
// 然后启动运行（如果你执行这个模式，用微信小程序打开会发现控制台有个报红，不会对程序有影响采用build会清除）
// or 执行构建的形式
npm run dev or npm run build
// 用微信小程序打开dist文件夹
```
